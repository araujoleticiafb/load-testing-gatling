# Load Testing with Gatling
For installation follow [https://gatling.io/open-source/](https://gatling.io/open-source/)

- Execute: 
```sh
$ brew  install scala
```
- Setting JDK

## Running 
 
1. Go to folder `bin` and execute:

- For Linux/MacOs: 
```sh
sh gathing.sh
```
- For Windows: 
```sh
gathing.bat
```

## Setting results
 Go to `conf/gathing.conf`
